# Personal Portfolio

A personal portfolio website designed to showcase my curriculum and technical skills. It serves as a central hub for my projects and includes a picture gallery to help visitors get to know me better.

## Overview
This project transforms a standard static site into an interactive experience using vanilla JavaScript. It is built to demonstrate clean code practices, DOM manipulation, and state management.

## Key Features
* **Curriculum & Skills:** A clear presentation of my professional background and technical abilities.
* **Dynamic Projects:** A JavaScript-driven gallery that allows users to filter my work by category.
* **Picture Gallery:** A dedicated section sharing personal photos and interests.
* **Interactive Elements:** Features a real-time validated contact form and a theme switcher that saves user preferences (Light/Dark/Blue).

## Technologies
* HTML5
* CSS3
* JavaScript
* Git

## How to Run
1.  Clone the repository:
    ```bash
    git clone [https://github.com/mavagoncalves/portfolio.git](https://github.com/mavagoncalves/portfolio.git)
    ```
2.  Open `index.html` in your browser.
3.  For the best experience (to ensure Cookies work correctly), use a local server like Live Server for VS Code.
